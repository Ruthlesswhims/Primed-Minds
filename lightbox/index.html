Introduction

light weight (14k minified), fast and easy to implement, content can be youtube, audio, etc.

Install

include the following in the head section

		
<head>
    <link rel="stylesheet" type="text/css" href="css/boom-lightbox/boom-lightbox.1.0.0.min.css" />
    <script type="text/javascript" src="js/boom-lightbox/boom-lightbox.1.0.0.min.js"></script>
</head>

Usage

Use the css class boom-lightbox on the container div that contains content


<body>
    <a href="#" onclick="boomLightBoxShow('divStatic');">Open Video in Static Lightbox</a>
    <div id="divStatic" class="boom-lightbox" style="display: none;">
    	<iframe id="ytplayer" width="720" height="405" src="https://www.youtube.com/embed/M7lc1UVf-VE" frameborder="0" allowfullscreen></iframe>
    </div>
    
    <a href="#" onclick="boomLightBoxShow('divResponsive');">Open Content in Responsive Lightbox</a>
    <div id="divResponsive" class="boom-lightbox" style="display: none;">
        <p>This is responsive content</p>
    </div>
    
    
    <!-- place the initialization at the end of the code -->
    <script type="text/javascript">
    	 (function() { //document ready, init one or more lightbox divs
            boomLightBoxInit([{
                selector: 'divStatic',
                configs: { width: '721px', height: '406px' }
            },{
                selector: 'divResponsive',
                configs: { width: '60%', height: '60%' }
            }]);
        })();
    </script>
</body>
